# Mr. Bean — Frontend

Follow the design system in `design-system/` for all UI work.

- **Tokens**: `design-system/colors_and_type.css` — colors, typography, spacing, radii, shadows, animation
- **Assets**: `design-system/assets/` — logo variants (default, icon-only, dark)
- **Reference**: `design-system/preview/` — HTML cards for colors, type, spacing, components, and brand
- **Prototype**: `design-system/ui_kits/app/index.html` — interactive mobile prototype (Home, Beans, Stats, Log Shot)
- **Brand & voice**: `design-system/README.md` — visual foundations, voice/tone, iconography (Lucide Icons)
